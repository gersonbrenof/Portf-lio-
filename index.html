Perguntas Frequentes (FAQ)

Quais são as principais tecnologias que você utiliza no Back-End?

Tenho conhecimento sólido no ecossistema Python com Django e Django REST Framework (DRF), principalmente focado no desenvolvimento de APIs de alta performance e integração de scripts/modelos de Inteligência Artificial. Também atuo no ecossistema Node.js utilizando o framework NestJS para arquiteturas modulares e escaláveis.

Você tem experiência prática com Inteligência Artificial?

Sim. Tenho forte experiência na integração de serviços de IA no Back-End. Isso inclui a orquestração e consumo de APIs externas (como Google Gemini, Groq Cloud e outros LLMs) para construir sistemas de recomendação e interações inteligentes. Além disso, também desenvolvo soluções locais, como uma API REST em Django acoplada a uma rede neural treinada para executar reconhecimento facial diretamente no servidor.

Como você aplica seus conhecimentos de DevOps no dia a dia?

Meu foco principal está em automação e cultura Full-Cycle. Garanto que o software seja integrado e entregue perfeitamente através de pipelines de CI/CD utilizando GitHub Actions, administrando servidores Linux (Ubuntu Server) e configurando ambientes baseados em Docker e nuvem (AWS).
